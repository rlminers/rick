Working Just on a Local Machine

git clone https://github.com/rlminers/rick.git

git config user.email "rlminers@gmail.com"
git config user.name "Rick Miners"

git pull - updates your repository
you may now edit as much as you like, save when you are done

git status (shows you which files you have modified)

git add FilesYouHaveModified (you want to git add every file you have modified during your editing)
git add .

git commit (to commit changes you have made to your local repository
git commit -m "Add existing file"

git push (will send all your edits to the repository hosted on github so that anyone else can pull those updates and continue working
